# NativeCheckout

## Docs

https://paypal.github.io/paypalnativecheckout-docs/iOS

## Installation

NativeCheckout is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'PayPal.Checkout'
```
